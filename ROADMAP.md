# Roadmap

Socializ is being built in public. This roadmap describes direction rather than a promise of release dates; priorities may change as the community and product evolve.

## Current focus

- Establish the modular Nx architecture and bounded-context boundaries.
- Build the web application shell and documentation site.
- Define the first identity, organization, access-control, content, and audit workflows.
- Improve test coverage and make the local development experience reproducible.

## Near-term

- Add persistent storage and migrations for the first production workflows.
- Implement authentication, organizations, roles, and permissions.
- Add API contract documentation and examples.
- Add reliable API and browser end-to-end coverage to CI.
- Publish the first development release and migration guidance.

## Later

- Expand social features such as profiles, posts, feeds, and moderation.
- Add observability, rate limiting, background jobs, and deployment references.
- Document extension points for integrations and community-built modules.

Have an idea or want to help? Open a focused issue or start a discussion before making a large change.
